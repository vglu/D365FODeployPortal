# Scripts

Run all scripts **from the project root**, e.g.:

```powershell
.\scripts\run.ps1
.\scripts\publish.ps1
.\scripts\check-prerequisites.ps1
.\scripts\Setup-ServicePrincipal.ps1
.\scripts\run-tests.ps1
```

Scripts that need paths to the solution or `src/` resolve the project root automatically (parent of `scripts/`).
